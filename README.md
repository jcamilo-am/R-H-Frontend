# R-H-Frontend
Frontend de Hotel Booking (Desarrollo de Apps 2)

# Tecnologías Usadas

- React 19
- Vite
- React Router DOM v7+
- Apollo Client
- Mantine UI

# Cómo ejecutar el proyecto

# 1. Instalar dependencias
npm install

# 2. Modo desarrollo
npm run dev

# 3. Compilar para produccion
npm run build
